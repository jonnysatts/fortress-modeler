# Fortress Modeler

This project is built with [Vite](https://vitejs.dev/), React, TypeScript, shadcn-ui and Tailwind CSS.

## Installation

Install dependencies using npm:

```sh
npm install
```

## Development

Start the development server with hot reload:

```sh
npm run dev
```

## Building for Production

Create an optimized build:

```sh
npm run build
```

The build output is generated in the `dist` directory. To preview the production build locally run:

```sh
npm run preview
```
