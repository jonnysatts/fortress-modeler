# Electron App Assets

This directory contains assets for the Electron desktop application.

## Required Files:

1. **icon.png** - App icon for Linux/macOS (512x512 px recommended)
2. **icon.ico** - App icon for Windows (multiple sizes: 16, 32, 48, 64, 128, 256 px)
3. **icon.icns** - App icon for macOS (if targeting macOS)

## Creating Icons:

You can create these from your main app logo using online converters:
- PNG to ICO: https://convertico.com/
- PNG to ICNS: https://cloudconvert.com/png-to-icns

## Recommended Icon Specifications:

- **Source image**: 1024x1024 px PNG with transparent background
- **Style**: Simple, recognizable at small sizes
- **Colors**: High contrast, readable
- **Format**: Square aspect ratio
